initialize
	- lag repository på gitlab.com 
	- kopiere navn og addressa i cmd
	- lag nokre filer
	- git init: i riktig mapappe
	- git add -A / relevant fil
	- git commit -m "melding"
	- git remote add orgin https://gitlab.com/atoft/test2.git
	- git push orgin master

lagg til fil
	- git add labTest3.py
	- git commit -m "melding"
	- git push orgin master

branching
	- git branch testBrach //oppret branch
	- git checkout testBranch
	- git add -A   |  evt.relevnat fil
	- git commit -m "melding"
	- git push orgin testBranch


merging
	- endre til master branch: git checkout master
	- git merge testBranch
	- gut push orgin master

slett branch
	- git branch -D testBranch
